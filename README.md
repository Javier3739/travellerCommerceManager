# Gestor de naves para Traveller

Este programa tiene como propósito facilitar el proceso de gestión del transporte y mercancías en Traveller, así como gestionar los ingresos de dichas actividades y gastos básicos, como combustible, mantenimiento e hipoteca de las naves.

## Versiones

Este programa estará disponible en tres versiones diferentes. Un sistema prototipo escrito en Python, una versión móvil escrita en Java, y una versión web. Esta última dará la capacidad a los narradores controlar y autorizar las distintas operaciones para los jugadores.
